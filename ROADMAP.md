# CareerOS Roadmap

CareerOS is entering its first public open-source release phase. The application foundation is stable enough for content work and community hardening.

## Now: Initial Public MVP

- Keep the current architecture stable.
- Improve documentation and contributor onboarding.
- Build a high-quality library of learning content.
- Fix bugs and accessibility issues in existing flows.
- Keep local-first behavior reliable.

## Before v1.0

- Expand published lessons across career tracks.
- Add content contribution review standards.
- Add automated tests for content parsing and critical UI flows.
- Review accessibility against WCAG 2.2 AA expectations.
- Add route-level code splitting if bundle size becomes a release blocker.
- Document deployment and release tagging workflow.

## Later

- Optional cloud sync behind repository adapters.
- Import and export workflows.
- Community content packs.
- Search across lessons and workspace data.
- Optional AI coaching integrations with clear privacy boundaries.

## Out of Scope for the MVP Release

- Major new product areas
- Interview-prep-only repositioning
- Mandatory authentication
- Cloud persistence as the default storage model
- Analytics or tracking that sends user data externally by default
