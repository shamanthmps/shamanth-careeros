# Changelog

All notable changes to CareerOS will be documented in this file.

The project follows semantic versioning after the first stable release.

## [0.1.0] - 2026-07-01

### Added

- Initial CareerOS MVP foundation.
- Dashboard experience for progress, missions, learning journey, and milestones.
- Local-first domain model, repository layer, and persistence adapter.
- Learning roadmap, module detail pages, and lesson pages.
- Offline Markdown content engine using `content/` lessons and YAML front matter.
- Open-source documentation, issue templates, pull request template, and CI checks.
