# Cloud Architecture

## Goal

Build interview-ready cloud architecture depth for Senior, Staff, and Principal TPM roles by covering infrastructure models, resilience design, security controls, scaling, and cost governance.

## Estimated Duration

9-12 hours

## Outcome

After this module, you should be able to:
- Explain cloud design trade-offs with senior-level clarity in interviews.
- Choose deployment and service models by workload, compliance, and team capability.
- Design resilient cloud systems with HA, DR, and multi-region strategies.
- Apply cost and governance controls to keep cloud architecture sustainable at scale.
