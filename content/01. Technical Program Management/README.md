# TPM Fundamentals

## Goal

Build strong technical program management fundamentals for driving complex, cross-functional initiatives with predictable delivery and clear business outcomes.

## Estimated Duration

3 to 4 hours

## Lessons

- 1. What Is a Technical Program Manager? (`01-what-is-a-tpm.md`)

## Outcome

By the end of this module, learners should be able to explain TPM scope, differentiate TPM from adjacent roles, and apply core execution concepts including alignment, dependency management, risk management, and executive communication.
