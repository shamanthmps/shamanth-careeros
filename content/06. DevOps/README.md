# DevOps

## Goal

Build interview-ready DevOps depth for Senior, Staff, and Principal TPM roles by covering delivery pipelines, reliability operations, security integration, and platform-scale governance.

10-14 hours

## Outcome

After this module, you should be able to:
- Explain DevOps architecture and execution trade-offs in interviews.
- Design CI/CD, release, and incident workflows with measurable reliability controls.
- Apply DORA-driven improvement planning across delivery organizations.
- Lead DevSecOps and platform engineering initiatives with risk-aware governance.
